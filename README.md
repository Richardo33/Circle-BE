Backend service for the Circle social app. Provides RESTful APIs for authentication, user profiles, threads, comments, and reactions, and is designed to work together with the Circle frontend project.

## Features

- User registration and login with JWT authentication  
- CRUD operations for threads and comments  
- Like / unlike functionality  
- Basic role-based authorization (adjust as needed)  
- PostgreSQL database using Prisma ORM  

## Tech Stack

- Node.js  
- Express.js  
- TypeScript  
- Prisma ORM  
- PostgreSQL  
- JWT for authentication  

## Getting Started

### Prerequisites

- Node.js LTS  
- PostgreSQL database  
- npm or yarn
